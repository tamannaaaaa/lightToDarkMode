# lightToDarkMode
Description
The Light to Dark Mode Website is a responsive web application that allows users to switch between light and dark themes with a simple toggle. It provides a comfortable viewing experience for users in different lighting conditions and personal preferences. The website is built using HTML, CSS, and JavaScript, making it easy to integrate into any project or website.

Features
Smooth transition between light and dark themes.
User's preference for theme mode is saved in the browser for future visits.
Mobile-responsive layout for optimal viewing on various devices.
